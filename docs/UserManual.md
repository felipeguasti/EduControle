
# Manual do Usuário do EquipReserve

## Introdução
Este manual fornece instruções sobre como utilizar o sistema EquipReserve para reservar equipamentos e espaços.

## Fazendo uma Reserva
- Acesse a página inicial.
- Escolha o recurso que deseja reservar.
- Preencha o formulário de reserva com os detalhes necessários.
- Confirme a reserva.
